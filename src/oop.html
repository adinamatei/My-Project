Object Oriented Programing in JavaScript

The basic idea of Object Oriented Programming (OOP) is that we use objects to model things from real life in our applications, and provide a simple way to access the functionality that would otherwise be hard to make use of.
Object can contain related data and code, which represent information about the things you are trying to model, and functionality or behavior that you want it to have.
    Building applications whit objects allows us to adopt some valuable techniques:
  • Inheritance -- objects can inherit characteristics from other objects
  • Encapsulation -- refers to enclosing (encapsulate) all the functionalities of an object inside that object so that methods and properties are hidden for the rest of application
  • Polymorphism -- different objects can share the same methods or properties, while their implementation may differ
  • Abstraction -- creating a simple model of a more complex thing, which represents its most important aspects in a way that is easy to work with for our program's purposes
